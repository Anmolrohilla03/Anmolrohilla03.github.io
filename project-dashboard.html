Understanding the Challenge

Building dashboards for different industries isn't just about placing charts and tables — it's about distilling complexity, prioritizing actions, and making sense of diverse, fast-moving data. Each domain brings unique workflows, KPIs, and user mindsets. My goal was to design unified yet context-aware dashboards that reduce cognitive load, surface what matters most, and remain scalable across teams.

Simplifying Hospital Oversight: A Unified Admin Dashboard

Hospital administrators often struggle with fragmented tools and disconnected data. I designed this dashboard to bring critical hospital operations into one centralized view, helping admins make faster, smarter decisions.

This dashboard combines metrics from all key navigation tabs—like Hospital Management, Security, RFID Infrastructure, and System Analytics —into one seamless interface.

What I Did

  • Conducted stakeholder interviews with hospital admins
  • Identified key KPIs from across 7 core admin functions
  • Designed a dashboard layout that surfaces the most important info in a single view
  • Focused on clarity, speed of understanding, and visual prioritization
  • Built using Pycube's design system to maintain accessibility and consistency

Highlighted Metrics

📈 Hospital performance trends
⚠️ Compliance violation rate
🔧 RFID hardware status
🚫 Unauthorized access attempts
⚡ System performance
🎫 Ticket resolution speed

Design System & Reusability

This helped streamline design workflows and enabled quick implementation by engineering teams across healthcare, finance, and procurement dashboards.
