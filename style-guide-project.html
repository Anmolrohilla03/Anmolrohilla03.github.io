At Pycube, I led the development of a unified design style guide to solve inconsistencies across the product ecosystem. I built a system of scalable foundations — typography, colors, components, tokens — and collaborated with developers to bring a robust React component library to life. Accessibility Considerations

  • Minimum body text size of 16px (1rem)
  • Line height ratio of at least 1.5 for optimal readability
  • Sufficient color contrast with background elements
  • Consistent vertical spacing for content scanning
  • Font used: Poppins (accessible sans-serif)

Design System Foundations

Design Tokens

Design tokens are the atomic elements of our design system — named values that store visual attributes like colors, typography, and spacing. By centralizing these values as tokens, we created a single source of truth that maintains consistency across platforms and enables rapid theming and customization.

Primitive Colors

Color tokens form the foundation of our visual language, providing a consistent and accessible palette across interfaces. They include primary brand colors, neutrals for UI structure, and status indicators for feedback.

primary-500: #01042E
neutral-200: #E6E6EA
accent-500: #805AD5

Primitive Typography

Typography tokens were created to establish a consistent typographic system across all platforms. These primitive tokens define font families, sizes, weights, and line heights. By centralizing these styles as variables, I ensured scalable and responsive type usage across the design system, enabling clarity, readability, and rapid reuse in components and layouts.

font-family: Poppins
font-size-md: 16px
font-weight-regular: Regular

Semantic Tokens

Semantic tokens connect visual attributes with functional meaning, creating a vocabulary that bridges design and development. They communicate intent rather than just appearance.

surface-primary: var(--primary-500)
alert-error: var(--red-500)
text-muted: var(--gray-600)

Using Variables2CSS, we automatically generate clean CSS variables from our design tokens, ensuring perfect translation from Figma to code. This streamlined process has significantly reduced development time and maintained consistency across our product ecosystem.
